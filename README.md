# my-skills

个人 AI 代理技能库集合 / Personal AI Agent Skills Collection

## 📚 技能列表 / Skills

### 1. 🧠 智能回答框架 / Smart Answering Framework
- **文件夹**: `answer-framework/`
- **描述**: 自动适配问题类型，提供有据可依的自然回答
- **特性**:
  - 智能问题分类
  - 多种回答模式（简洁、详细、对比、观点等）
  - 双语支持
  - 严谨的推理和论证

### 2. ✅ 双重验证技能 / Double-Check Skill (Code Edition)  
- **文件夹**: `double-check-code/`
- **描述**: 代码修改后自动进行两次独立验证，确保无误
- **特性**:
  - 自动触发验证流程
  - 两次连续验证
  - 支持编译、测试、运行检查
  - 适用于所有代码修改场景

## 🚀 使用方式 / Usage

每个技能都包含详细的使用说明和实现指南。请查看各技能文件夹中的 `SKILL.md` 文件了解更多细节。
