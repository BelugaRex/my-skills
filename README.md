# my-skills

个人 AI 代理技能库集合 / Personal AI Agent Skills Collection

## 📚 技能列表 / Skills

### 1. 🧠 智能回答框架 / Smart Answering Framework
- **文件夹 / Folder**: `answer-framework/`
- **描述 / Description**: 自动适配问题类型，提供有据可依的自然回答 / Adapts to question types and delivers evidence-based, natural responses.
- **特性 / Highlights**:
  - 智能问题分类 / Smart question classification
  - 多种回答模式（简洁、详细、对比、观点等） / Multiple response modes (concise, detailed, comparison, opinion, etc.)
  - 双语支持 / Bilingual support
  - 严谨的推理和论证 / Rigorous reasoning and argumentation

### 2. ✅✅ 双重验证技能 / Double-Check Skill  
- **文件夹 / Folder**: `double-check/`
- **描述 / Description**: 修改任何文件后自动进行两次独立验证，确保无误 / Automatically performs two independent validations after any file change to ensure correctness.
- **特性 / Highlights**:
  - 自动触发验证流程 / Automatically triggers the validation flow
  - 两次连续验证 / Two consecutive validations
  - 支持编译、测试、运行、文档/配置检查 / Supports compile, test, run, doc/config checks
  - 适用于所有文件修改场景 / Applies to all file modification scenarios
  - 通过 ✅✅ 激活标识提示技能已调用 / Uses ✅✅ as the activation marker
